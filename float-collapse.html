All About Floats

A Float is a CSS positioning property whose properties can be traced to print design. In the pages of a magazine, for example, text will wrap around an image as needed. In web design, elements with a float property applied are just like images in a print layout. Float collapse occurs when a parent element contains nothing but floated elements. This results in the literal collapse of its height.
