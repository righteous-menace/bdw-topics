What Is Clearfix?

Clearfix is usually used in float layouts as a way to allow an element to automatically clear itself without adding additional markup. An interesting sidenote, IE usually requires extra markup.

A New Micro Clearfix Hack

The clearfix hack is a way to contain floats without the need for extra code by using the :before and :after pseudo-elements. The :before pseudo-element prevents top-margin collapse and the :after pseudo-element clears the floats. Using these “micro clearfixes,” the total amount of code is reduced.
